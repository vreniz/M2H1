# 🌐 Portafolio Web — Vanessa Fontalvo

Este proyecto consiste en un portafolio personal desarrollado con **HTML5 y CSS3 moderno**, enfocado en buenas prácticas de semántica, diseño responsivo y experiencia de usuario.

Incluye:
- Página principal (portafolio)
- Página secundaria (mascotas)
- Hoja de estilos compartida

# 🏷️ 2. Etiquetas HTML utilizadas

## 📌 Estructura semántica principal

Se usaron etiquetas semánticas modernas de HTML5:

- <header> → Encabezado con navegación
- <nav> → Menú de navegación
- <main> → Contenido principal
- <section> → División por secciones
- <article> → Tarjetas (ej: mascotas)
- <footer> → Pie de página

---

## 📌 Etiquetas de contenido

- <h1>, <h2>, <h3> → Jerarquía de títulos
- <p> → Párrafos
- <span> → Elementos inline decorativos
- <em> → Énfasis semántico

---

## 📌 Listas

- <ul> → Listas no ordenadas
- <li> → Elementos de lista

---

## 📌 Enlaces

- <a> → Navegación interna y externa

## 📌 Formularios
